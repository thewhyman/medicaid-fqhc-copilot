Agent Architecture

Six specialized AI agents, each independently testable, coordinated by a central orchestrator:

Risk Scoring Agent Prioritizes patients by likelihood of coverage loss
Outreach Agent TCPA-compliant SMS/voice communication sequences
Conversation Agent Multilingual Q&A, guidance, and instructions
Document Intelligence Parses, validates, and verifies uploaded documents
Workflow Orchestrator Manages state transitions and triggers next actions
Caseworker Copilot Surfaces insights and suggests actions

Evaluation-first architecture: every agent action is measured. Deterministic guardrails catch errors before they reach patients.
